MERN Authentication Project
This is a simple authentication system built using the MERN stack (MongoDB, Express, React, and Node.js).

Features
User registration
User login
JWT authentication
Protected routes
Prerequisites
Make sure you have the following installed on your system:

Node.js
npm or yarn
MongoDB
